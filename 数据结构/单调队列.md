# 单调队列

用于查询一段连续子数组中的最大值或最小值

##  模板

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 1e5 + 10;
int V[N]; // 存储的是数值
int n, m; // n个数，长度为m的窗口

int main()
{
    //滑动窗口,单调队列
    deque<int> Q; // 存储的是编号
    for (int i = 0; i < n; ++i)//i循环的是窗口头到达了哪一个位置,m-1时开始输出
    {
        if (!Q.empty() && i - Q.front() >= m) // 判断队头是否已经超出了窗口
            Q.pop_front();
        while (!Q.empty() && V[Q.back()] < V[i]) // 比后面将要加入的元素小，则弹出.因为窗口头已经循环到了这里,前面的元素已经不可能成为最大值了
            Q.pop_back();
        Q.push_back(i); // 新生入队
        if (i >= m - 1)
            cout << V[Q.front()] << " ";
    }
}
```
